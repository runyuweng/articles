# 生成manifest
一旦修改代码，打包是自动更新混存清单：
```
webpackConfig.plugins.push(
  new AppCachePlugin({
    cache: [
      'common.js',
      'index.js',
      'index.css',
    ],
    network: ['*'],  // No network access allowed!
    //fallback: ['index.html'],
    settings: ['prefer-online'],
    //exclude: [],  // Exclude file.txt and all .js files
    output: 'appcache.manifest'
  }));
```
